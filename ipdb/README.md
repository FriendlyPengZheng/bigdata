# ipdb-java
IPIP.net officially supported IP database ipdb format parsing library

# Maven 
<pre>
Added to pom.xml

&lt;dependencies&gt;
    &lt;dependency>
        &lt;groupId>net.ipip&lt;/groupId&gt;
        &lt;artifactId>ipdb&lt;/artifactId&gt;
        &lt;version>1.1.2&lt;/version&gt;
    &lt;/dependency&gt;
&lt;/dependencies&gt;
</pre>
